# Entity Functions

## entity

**entity**(type, key)

Create an entity of the given type

## entityId

**entityId**(entity)

Returns the entity id of the entity

## entityList

**entityList**(type, keys)

Create a list of entities of the given type

## uniqueId

**uniqueId**(entity)

Returns the unique id of the entity as a string
